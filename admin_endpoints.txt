Base URL: http://localhost:8989

Authentication:
- Use Basic Auth for authenticated calls (username/password).
- Login endpoint (/api/users/login) is public.

Login response format:
{
  "message": "...",
  "status": true|false,
  "role": "ADMIN|APPROVER|POLICYHOLDER|null"
}

General API response format:
{
  "message": "...",
  "status": true|false,
  "data": ...
}

---
ADMIN Endpoints
---

1) Login
POST /api/users/login
Body (JSON):
{
  "username": "admin",
  "password": "adminPassword"
}
Expected: 200, status true, role = ADMIN

2) Users
GET /api/users
- Fetch all users

GET /api/users/{userId}
- Fetch single user

GET /api/users/role/{role}
- Example: /api/users/role/APPROVER

POST /api/users
Body example:
{
  "username": "newuser",
  "password": "secret",
  "fullName": "New User",
  "email": "newuser@example.com",
  "phoneNumber": "9999999999",
  "dateOfBirth": "1990-01-01",
  "address": "123 Main St",
  "role": "POLICYHOLDER",
  "accountStatus": "ACTIVE",
  "bankAccountNumber": "1234567890",
  "ifscCode": "BANK0001",
  "bankName": "Test Bank"
}
Expected: 201, data = created user

PUT /api/users/{userId}
Body example: full update payload like POST body
Expected: 200, data = updated user

PATCH /api/users/{userId}
Body example: partial update
{
  "phoneNumber": "8888888888",
  "accountStatus": "INACTIVE"
}
Expected: 200, data = patched user

DELETE /api/users/{userId}
Expected: 200 when deleted, 404 if missing
